## Time Audit Data Normalization
*From AIP* Reads web and browser data, converts timestamps into a standard format for analysis, adjusts the time zone from UTC to EST or America/New_York, and drops unnecessary columns. Finally, it combines the data from both sources and extracts specific details like the day of the week and hour from the end timestamp for further insights.

## Domain Table Pipeline
*From AIP* Reads raw time usage data and summarizes it by finding the most recent activity time for each domain and counting the number of distinct starting times. It then classifies each domain into "Productive" or "Not Productive" based on user interaction, and further categorizes each domain into specific categories such as Career, Communication, or Entertainment.

## Time Audit Finale
*From AIP* Reads data from two sources: one containing information about categories of domains and the other related to time audits of app usage. It then filters the time audit data to exclude entries from a specific application. Finally, it combines the filtered time audit data with the domain category information by linking them based on matching domain names, while keeping specific details from both datasets for further analysis.
